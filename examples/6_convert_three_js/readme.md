Как создать viewlang-объект из threejs-сущности

1. Создайте файл объекта на основе MyLoader.qml 
2. Подключите его в какую-нибудь сцену.

Примеры подключения

* test1-simple.vl Простое подключение
* test2-complex.vl Подключение ThreeJS-объекта + параметры
* test3-fun-timer.vl Параметры + таймер
* test4-fun-jitter.vl Изменение геометрии по таймеру
* test5-repeater.vl Динамическое создание компонент (через Repeater)
